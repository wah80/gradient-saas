Gradient API

Use our free API to generate random CSS gradients for your applications.

Endpoint

GET /api/random-gradient

Returns a random CSS gradient.

Example Request

GET https://127.0.0.1/api/random-gradient

Example Response

{
  "type": "linear",
  "angle": 120,
  "colors": ["#ff6a00","#ee0979"],
  "css": "background: linear-gradient(120deg,#ff6a00,#ee0979);"
}

JavaScript Example

fetch("/api/random-gradient")
.then(res => res.json())
.then(data => {
    console.log(data.css);
});
